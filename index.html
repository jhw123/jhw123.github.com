automaTA

Human-machine Collaboration for QnA Automation
Sept. 2018 ~ April 2019
Our team for CS project course developed a QnA system that answers programming beginners' questions on function usage. With 14 participants we conducted two experiments and found that the system is more effective to a function search than online searches. Our team summarized the results and submitted a paper to L@S 2019 Work-in-progress.

My Role

  • Designed and developed the QnA window in a form of Chrome extension
  • Implemented a server notification feature on Chrome through background.js
  • Developed a prototypical coding platform by using React.js and Semantic UI library
  • Evaluated the system with a A/B testing

Tech Stack

Chrome extension, React.js, node.js, AWS, Firebase Realtime DB
